# BioLinux: 바이오 분야 취업 준비생을 위한 리눅스 입문 교재

이 저장소는 **리눅스를 처음 접하는 바이오 분야 취업 준비생**을 위한 기초 교재입니다.  
바이오 데이터 분석 업무에서 자주 마주치는 리눅스 명령어, 파일 처리, 원격 서버 사용, 개발 환경 구성 실습을 중심으로 구성되어 있습니다.


## 대상

- 리눅스를 처음 접하는 **바이오 전공자 및 바이오 분야 취업 준비생**
- 연구실, 분석 서버, 클라우드/원격 서버 환경에서 기본적인 파일 처리와 작업 실행이 필요한 학습자


## 구성

본 교재는 **5일 과정의 실습 중심 커리큘럼**으로 구성되어 있습니다.

| Day | 주제 |
|-----|------|
| Day 1 | 리눅스 기초 및 파일 시스템 구조 이해 |
| Day 2 | 파일과 프로세스 관리, 텍스트 편집 |
| Day 3 | 시스템 관리 및 기본 네트워크 |
| Day 4 | 환경 설정 및 도구 설치 |
| Day 5 | 실무 환경 구성 및 확장 |

교재 사이트: [https://seondh.github.io/bio_linux/](https://seondh.github.io/bio_linux/)


## 사용 환경

- Ubuntu 22.04 LTS 기준
- 터미널 환경 기반 실습
- 원격 서버 접속, Docker, SLURM 등은 실무에서 자주 접하는 수준의 기본 사용법 중심
