# [번외] cowsay

### `cowsay`

1. **기본 사용법**:
    
    ```bash
    cowsay "패키지 관리 실습 중입니다!"
    ```
    
2. **다른 동물 사용**:
`cowsay`에는 소 이외에도 여러 동물이 포함되어 있다. `f` 옵션을 사용하여 다른 동물로 변경할 수 있다.
    
    ```bash
    cowsay -f tux "Hello from Tux!"
    ```
    
3. **`fortune`과 함께 사용**
`fortune` 패키지를 설치하면 랜덤한 명언이나 재미있는 문구를 `cowsay`와 함께 사용할 수 있다.
    
    ```bash
    sudo apt install fortune
    fortune | cowsay
    ```
