# [번외] 비밀번호 변경하기

리눅스에서는 `passwd` 명령어로 계정 비밀번호를 변경할 수 있다. SSH 접속 계정을 사용할 때는 비밀번호를 주기적으로 바꿔 두는 습관이 좋다.

## 1. 현재 사용자 비밀번호 변경

```bash
passwd
```

현재 로그인한 사용자의 비밀번호를 변경한다.

## 2. 다른 사용자 비밀번호 변경

```bash
sudo passwd username
```

관리자 권한이 있으면 다른 사용자의 비밀번호도 변경할 수 있다.

## 3. 확인

비밀번호를 변경한 뒤에는 SSH 접속을 다시 시도해서 정상적으로 로그인되는지 확인한다.

## 요약

1. `passwd`는 현재 사용자의 비밀번호를 변경한다.
2. `sudo passwd username`은 다른 사용자의 비밀번호를 변경한다.
3. 비밀번호를 바꾼 뒤에는 SSH 접속을 다시 확인한다.
