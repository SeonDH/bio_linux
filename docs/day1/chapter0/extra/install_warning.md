# [번외] 설치 페이지 광고와 피싱 주의

소프트웨어를 설치할 때는 검색 결과 상단에 보이는 광고 링크를 바로 누르지 않는 습관이 중요하다.  
광고는 겉으로는 공식 사이트처럼 보이더라도, 실제로는 비슷한 이름의 가짜 사이트로 연결될 수 있다.

특히 설치 명령을 복사해서 터미널에 붙여 넣는 방식은 편리하지만, 잘못된 사이트에서 받은 명령을 실행하면 악성 코드가 설치될 수 있다.  
Homebrew처럼 자주 사용하는 도구는 이런 방식의 공격 대상이 되기 쉽다.

다음 원칙을 기억한다.

- 검색 결과의 광고 표시를 먼저 확인한다.
- 주소창의 도메인이 공식 사이트와 정확히 일치하는지 본다.
- 설치 페이지는 검색 엔진보다 공식 문서 링크나 즐겨찾기를 우선한다.
- 의심스러우면 주소를 직접 입력해서 접속한다.
- 설치 명령은 실행하기 전에 출처를 한 번 더 확인한다.

예를 들어 Homebrew는 공식 사이트가 `brew.sh`이므로, 설치가 필요하면 검색 결과를 거치지 말고 공식 주소를 직접 연다.

이런 점검 습관은 Homebrew뿐 아니라 Python, Docker, 패키지 관리자, 각종 CLI 도구를 설치할 때도 그대로 적용된다.
