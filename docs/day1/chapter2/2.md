## 설치 후 첫 화면 설명

- `Ubuntu 24.04 LTS seon-server tty1`
  - 현재 Ubuntu 24.04 LTS를 사용 중
  - 호스트명은 `seon-server`
  - 현재 로그인 터미널은 `tty1`

- 기본 정보
  - System load: CPU 사용률
  - Usage of `/`: 디스크 사용률
  - Memory Usage: 메모리 사용률
  - Swap Usage: 디스크 기반 임시 메모리 사용률
  - Processes: 현재 실행 중인 프로세스 수
  - Users logged in: 로그인된 사용자 수
  - IP address (IPv4/IPv6): enp0s1 기준

- 프롬프트:
  ```shell
  seon@seon-server:~$
  ```
