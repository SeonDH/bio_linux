# [답지] 범위 지정 후 편집

```bash
vi practice3.txt
```

대표 명령 예시:

```text
:1,3d          1행부터 3행까지 삭제
:2,4y          2행부터 4행까지 복사
G
p              복사한 줄을 파일 아래에 붙여넣기
:.,$c          현재 행부터 마지막 행까지 변경
New content
New content
New content
Esc
:.,+2>         현재 행부터 2행 아래까지 들여쓰기
:.,-2<         현재 행부터 2행 위까지 내어쓰기
V
G
U              현재 행부터 마지막 행까지 선택한 뒤 대문자로 변경
:1,.s/foo/bar/g 1행부터 현재 행까지 foo를 bar로 변경
```

실습 파일에서 현재 행 위치에 따라 범위는 달라질 수 있다.
