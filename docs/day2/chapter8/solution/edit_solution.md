# [답지] vi 문서 편집

```bash
vi practice2.txt
```

대표 명령 예시:

```text
r문자          커서 위치의 글자 1개 바꾸기
cw             커서 위치의 단어 바꾸기
c2w            단어 2개 바꾸기
2s             글자 2개 바꾸기
cc             현재 행 전체 바꾸기
C              커서 위치부터 행 끝까지 바꾸기
x              글자 1개 삭제
2x             글자 2개 삭제
dw             단어 1개 삭제
d2w            단어 2개 삭제
dd             현재 행 삭제
2dd            현재 행부터 2줄 삭제
D              커서 위치부터 행 끝까지 삭제
u              마지막 명령 취소
Ctrl+r         취소한 명령 되돌리기
U              현재 행 변경 사항 되돌리기
:e!            마지막 저장 상태로 되돌리기
yy             현재 행 복사
2yy            현재 행부터 2줄 복사
p              아래쪽에 붙여넣기
P              위쪽에 붙여넣기
dd             현재 행 잘라내기
2dd            현재 행부터 2줄 잘라내기
```
