# [답지] vi 화면 모드 변경 및 화면 이동하기

```bash
vi practice1.txt
```

대표 명령 예시:

```text
$ 또는 A        행의 맨 마지막으로 이동
A456           행 끝에 456 입력
Esc            명령 모드로 돌아오기
o              다음 행을 만들고 입력 모드로 전환
:set number    행 번호 켜기
H              화면 맨 위로 이동
M              화면 중간으로 이동
L              화면 맨 아래로 이동
:60            60번 행으로 이동
0              행 처음으로 이동
$              행 마지막으로 이동
w              다음 단어로 이동
b              이전 단어로 이동
Ctrl+u         화면 위쪽 절반으로 이동
Ctrl+d         화면 아래쪽 절반으로 이동
G              파일 끝으로 이동
:wq            저장 후 종료
```
