## [실습] 검색 명령 키

아래 텍스트에서 검색 명령을 순서대로 실행한다.

- AS-IS

This is a foo test.
Another foo line.
Yet another foo example.

1. `/foo`를 입력해 아래 방향으로 검색한다.
   - 첫 번째 `foo` 위치로 이동한다.

2. `n`을 입력해 다음 검색 결과로 이동한다.
   - 다음 `foo` 위치로 이동한다.

3. `N`을 입력해 이전 검색 결과로 이동한다.
   - 이전 `foo` 위치로 이동한다.

4. `?foo`를 입력해 위 방향으로 검색한다.
   - 현재 위치보다 위쪽에 있는 `foo`를 찾는다.

## [실습] 바꾸기 명령 키

1. 현재 행에서 첫 번째로 나오는 'foo'를 'bar'로 바꾼다.
- 명령어: `:s/foo/bar`
- AS-IS
foo is a foo test with another foo.
- TO-BE
bar is a foo test with another foo.

2. 현재 행에서 모든 'foo'를 'bar'로 바꾼다.
- 명령어: `:s/foo/bar/g`
- AS-IS
foo is a foo test with another foo.
- TO-BE
bar is a bar test with another bar.

3. 파일 전체에서 모든 'foo'를 'bar'로 바꾼다.
- 명령어: `:%s/foo/bar/g`
- AS-IS
foo is a foo test with another foo.
Another foo line has foo twice.
Yet another foo example has foo again.
- TO-BE
bar is a bar test with another bar.
Another bar line has bar twice.
Yet another bar example has bar again.

4. (3번 내용을 undo 한다 `u` 명령어) 1행부터 2행까지 범위 내 각 행의 첫 번째로 나오는 'foo'를 'bar'로 바꾼다.
- 명령어: `:1,2s/foo/bar`
- AS-IS
foo is a foo test with another foo.
Another foo line has foo twice.
Yet another foo example has foo again.
- TO-BE
bar is a foo test with another foo.
Another bar line has foo twice.
Yet another foo example has foo again.

5. (4번 내용을 undo 한다 `u` 명령어) 1행부터 2행까지 범위 내 모든 'foo'를 'bar'로 바꾼다.
- 명령어: `:1,2s/foo/bar/g`
- AS-IS
foo is a foo test with another foo.
Another foo line has foo twice.
Yet another foo example has foo again.
- TO-BE
bar is a bar test with another bar.
Another bar line has bar twice.
Yet another foo example has foo again.

6. (5번 내용을 undo 한다 `u` 명령어) 1행부터 3행까지 범위 내 모든 'foo'를 'bar'로 바꿀 때 수정 여부를 묻는다.
- 명령어: `:1,3s/foo/bar/gc`
- AS-IS
foo is a foo test with another foo.
Another foo line has foo twice.
Yet another foo example has foo again.
- TO-BE
bar is a bar test with another bar. (사용자가 변경 확인)
Another bar line has bar twice. (사용자가 변경 확인)
Yet another bar example has bar again. (사용자가 변경 확인)
