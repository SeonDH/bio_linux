# [실습] 필터 - 10분

## 실습 전 준비

```bash
cat <<EOL > animals.txt
dog,mammal,4
cat,mammal,4
sparrow,bird,2
goldfish,fish,0
cat,mammal,4
dog,mammal,4
parrot,bird,2
dolphin,mammal,0
goldfish,fish,0
sparrow,bird,2
parrot,bird,2
EOL
```

## 실습 단계

1. "cat"을 포함하는 모든 줄을 출력하자 

2. "mammal"을 "MAMMAL"로 치환하여 출력하자

3. 첫 번째 필드만 출력하자

4. 정렬해서 출력하자
