# [실습] 포그라운드 / 백그라운드 - 5분

## 실습 전 준비

```bash
$ sleep 100
```


## 실습 단계

1. 실행 중인 **포그라운드 작업을 일시 중지** 해보자  
   - 힌트: 실행 도중 `Ctrl + Z` 입력

2. **작업 목록을 확인**해보자  
   - 힌트: `jobs` 명령어 사용

3. **일시 중지된 작업을 백그라운드에서 계속 실행**해보자  
   - 힌트: `bg` 명령어 사용

4. **작업 목록을 다시 확인**해보자
   - 힌트: `jobs` 명령어 사용
   - `Running` 상태로 보이면 백그라운드에서 계속 실행 중이다.

5. **백그라운드 작업을 포그라운드로 가져오기**  
   - 힌트: `fg` 명령어 사용

6. **포그라운드 작업을 종료해보자**  
   - 힌트: `Ctrl + C` 입력

> `bg`는 현재 터미널 세션에서 일시 중지된 작업을 다시 실행하는 명령이다. 터미널을 닫은 뒤에도 작업을 유지하는 목적이라면 `nohup` 실습을 사용한다.
