# [실습] 포그라운드 / 백그라운드 - 5분

## 실습 전 준비

```bash
$ sleep 100
```

---

## 실습 단계

1. 실행 중인 **포그라운드 작업을 일시 중지** 해보자  
   - 힌트: 실행 도중 `Ctrl + Z` 입력

2. **작업 목록을 확인**해보자  
   - 힌트: `jobs` 명령어 사용

3. **일시 중지된 작업을 백그라운드에서 계속 실행**해보자  
   - 힌트: `bg` 명령어 사용

4. **백그라운드 작업을 포그라운드로 가져오기**  
   - 힌트: `fg` 명령어 사용

5. **포그라운드 작업을 종료해보자**  
   - 힌트: `Ctrl + C` 입력
