# [실습] rsync - 5분

## 실습 전 준비

```bash
mkdir ~/rsync
cd ~/rsync
```


## 실습 문제

1. 로컬 PC에 `from_local1` 파일을 생성한다.
2. 다음 명령어를 이용하여 원격 서버와 디렉터리를 동기화한다. (동기화 후 서버의 파일 상태 확인)
3. 로컬 PC에 `from_local2` 파일을 추가로 생성한다.
4. 다시 위의 `rsync` 명령어를 사용하여 디렉터리를 동기화한다. (동기화 후 서버의 파일 상태 확인)
